


A ideia da demonstração por indução consiste na analgia com uma cadeia de dominós. Imagine que alguns dominós estejam enfileirados, mesmo que sejam infinitos dominós eu consigo provar que caso eu derrube 1 eventualmente derrubarei o último.

Da mesma forma, a ideia consiste em você provar através de 3 passos:
1) Provar um caso base P(0), P(1), ...
2) Provar um caso "k" P(k)
3) Provar um caso "k+1", em que através de análises e substituindo o P(k) tendemos a achar a expressão de k+1 que supomos inicialmente.


Exemplos [Gabriel Coutinho Vídeo](https://youtu.be/u1837tBWclA):

Teorema: Seja $a_n$ uma sequência definida por:

$$a_n = \sum_{i=1}^{n} i^2$$
Então, $a_n = \frac{(n)(n+1)(2n+1)}{6}$ .

Demonstração por indução.

Resolvendo:

Primeiro vou focar em resolver o caso base: P(1) = $a_1=\frac{(1)(1+1)(2*1+1)}{6}=1$. Isso bate com a ideia de somatório ao quadrado que foi fornecido pelo enunciado. (provado)

Agora, focar no caso "k": P(k) = $a_k =1+ 2^2+ 3^2 + 4^2 + ... + k^2 =\frac{(k)(k+1)(2*k+1)}{6}$. (afirmo)

Por fim, começo a pensar no caso "k+1": P(k+1) = $a_{k+1} =1+2^2+3^2+4^2+...+k^2+(k+1)^2 = \frac{(k+1)(k+1+1)(2*(k+1)+1)}{6}$

$a_{k+1} = \textcolor{green}{1+2^2+3^2+4^2+...+k^2}+(k+1)^2 = \frac{(k+1)(k+1+1)(2*(k+1)+1)}{6}$

Isso de verde, já vimos afirmado em P(k), logo substituindo:

$a_{k+1} = \textcolor{green}{\frac{(k)(k+1)(2*k+1)}{6}}+(k+1)^2 = \frac{(k+1)(k+1+1)(2*(k+1)+1)}{6}$

$a_{k+1} = \frac{(k)(k+1)(2*k+1)+6(k+1)^2}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(k^2+k)(2*k+1)+6(k+1)^2}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(2k^3+k^2+2k^2+k)+6(k+1)^2}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(2k^3+3k^2+k)+6(k+1)^2}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(2k^3+3k^2+k)+6(k^2+2k+1)}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(2k^3+3k^2+k)+(6k^2+12k+6)}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(2k^3+9k^2+13k+6)}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

Fatorando (achei difícil) chegamos exatamente na mesma expressão:

$a_{k+1} = \frac{(k+1)(k+2)(2k+3)}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$ (Provado por Indução)

Verificando a resolução do Gabriel Coutinho ele visualizou algo que eu não tinha visto. Colocar o k+1 em evidência:

$a_{k+1} = \frac{(k+1)((k)(2*k+1)+(6k+6))}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(k+1)(2*k^2+k+6k+6)}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

$a_{k+1} = \frac{(k+1)(2*k^2+7k+6)}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$

Por baskara: k1 = -2 k2=-3/2, logo:

$a_{k+1} = \frac{(k+1)(k+2)(2k+3)}{6} = \frac{(k+1)(k+2)(2k+3)}{6}$ (Provado por Indução)

Exemplo 2 do vídeo do Gabriel Coutinho:

Teorema: Para todo $n$ inteiro positivo, 5 divide $n^5 - n$.

Demonstração por indução:
Primeiro vamos testar o caso base: P(1)= $1^5-1=0 / 5= 0$
		                        P(2)= $2^5-2=30 / 5= 6$ (provado)

Agora, para o caso k: P(k) = $\frac{(k^5 - k)}{5} = inteiro qualquer$

Agora, o caso k+1: P(k+1) = $\frac{((k+1)^5 - (k+1))}{5}$ = $\frac{((k+1)^2(k+1)^2k - (k+1))}{5}$

P(k+1) = $\frac{((k^2+2k+1)(k^2+2k+1)(k+1) - (k+1))}{5}$

P(k+1) = $\frac{((k^4 + 2k^3 + k^2 + 2k^3 + 4k^2 + 2k + k^2 + + 2k + 1)(k+1) - (k+1))}{5}$

P(k+1) = $\frac{((k^4 + 4k^3 + 6k^2 + 4k + 1)(k+1) - (k+1))}{5}$

P(k+1) = $\frac{((k^5 + 4k^4 + 6k^3 + 4k^2 + 1k + k^4 + 4k^3 + 6k^2 + 4k + 1) - (k+1))}{5}$

P(k+1) = $\frac{((k^5 + 5k^4 + 10k^3 + 10k^2 + 4k)}{5}$ (isso num da, calma q vamos precisar do P(k))

logo ao invés de juntar tudo, vamos manter um termo q forme o P(k)=$(k^5 - k)$.

P(k+1) = $\frac{(k^5 + 5k^4 + 10k^3 + 10k^2 + 5k + 1 -k -1)}{5}$

P(k+1) = $\frac{(\textcolor{green}{k^5-k} + 5k^4 + 10k^3 + 10k^2 + 5k)}{5}$

Note que o termo de ver é o P(k) q vimos ser um divisor de 5, logo se o termo em ver é divido por 5 e os demais termos são múltiplos de 5 (5 e 10 como coeficientes) 5 divide n^5 -n. Está provado por indução!!
